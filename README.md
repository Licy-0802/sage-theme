# 🌿 Sage · 鼠尾草护眼主题

> **AI 时代，少写代码，多思考。**
> *Less typing, more thinking. A calming theme for the AI era.*

---

AI 让写代码这件事变了——不再逐字敲打，更多是阅读、审查、思考。Sage 是一个专为这种新工作方式设计的主题：低饱和鼠尾草绿底色，温柔护眼，让你在屏幕前待一整天也不累。

AI is changing how we code. Less typing, more reading, reviewing, and thinking. Sage is built for this new workflow — soft sage-green tones that are proven to reduce eye strain during long sessions.

---

## 🎨 配色 Palette

| 元素 | 色值 | 
|------|------|
| 背景 | `#f0f4e8` |
| 文字 | `#4a5d42` |
| 强调 | `#7c9a6b` |
| 关键字 | `#5A7D6B` **bold** |
| 字符串 | `#b8956a` |
| 方法 | `#7a6b5a` |
| 注释 | `#a3b899` *italic* |
